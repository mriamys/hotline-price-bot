# 🤖 hotline-price-bot

Telegram-бот для отслеживания цен на товары с [Hotline.ua](https://hotline.ua). Бот уведомляет тебя, когда цена на выбранный товар изменяется.

## ✨ Возможности

- 🔍 Поиск товаров на Hotline.ua прямо из Telegram
- 🔔 Подписка на отслеживание цены любого товара
- 📉 Уведомления о снижении цены
- 📈 Уведомления о повышении цены
- 🗂 Просмотр и управление подписками
- ⏱ Автоматическая проверка цен каждые 10 минут

## 🛠 Технологии

- [aiogram 3](https://docs.aiogram.dev/) — Telegram Bot framework
- [Playwright](https://playwright.dev/python/) — headless-браузер для парсинга
- [APScheduler](https://apscheduler.readthedocs.io/) — планировщик задач
- SQLite — база данных для хранения подписок

## 📁 Структура проекта

```
hotline-price-bot/
├── main.py            # Точка входа, хендлеры бота
├── hotline_parser.py  # Парсер Hotline.ua (поиск + цены)
├── checker.py         # Планировщик проверки цен
├── database.py        # Работа с SQLite
└── users_data.db      # База данных (создаётся автоматически)
```

## 🚀 Запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/hotline-price-bot.git
cd hotline-price-bot
```

### 2. Установи зависимости

```bash
pip install aiogram apscheduler playwright
playwright install chromium
```

### 3. Укажи токен бота

В файле `main.py` замени значение `TOKEN`:

```python
TOKEN = "ВАШ_ТОКЕН_ОТ_BOTFATHER"
```

> Получить токен можна у [@BotFather](https://t.me/BotFather)

### 4. Запусти бота

```bash
python main.py
```

## ⚙️ Настройка интервала проверки

По умолчанию бот проверяет цены каждые 10 минут. Изменить можно в `main.py`:

```python
scheduler.add_job(check_prices, 'interval', seconds=600, args=[bot])
#                                                      ↑
#                                             600 сек = 10 минут
```

## 📝 Примечания

- Бот использует headless-браузер (Chromium), поэтому требует чуть больше ресурсов, чем обычные парсеры
- Hotline.ua может менять структуру страниц — в таком случае нужно обновить CSS-селекторы в `hotline_parser.py`
- Не запускай слишком частые проверки, чтобы не получить бан по IP

## 🔒 Безопасность

### Защити токен бота

Токен бота — это как пароль. Если он попадёт в публичный репозиторий, любой человек сможет управлять твоим ботом от твоего имени. GitHub даже автоматически сканирует репозитории на утечку токенов, и Telegram может его отозвать.

**Что делать:** хранить токен в отдельном файле `.env`, который не попадает в Git.

**Шаг 1.** Установи библиотеку для работы с `.env`:

```bash
pip install python-dotenv
```

**Шаг 2.** Создай файл `.env` в корне проекта:

```
BOT_TOKEN=сюда_вставь_свой_токен
```

**Шаг 3.** Измени `main.py` — убери токен из кода:

```python
from dotenv import load_dotenv
import os

load_dotenv()
TOKEN = os.getenv("BOT_TOKEN")
```

**Шаг 4.** Убедись, что `.env` добавлен в `.gitignore` (см. ниже) — тогда он никогда не попадёт в репозиторий.

### Если токен уже утёк

Если ты случайно запушил токен в публичный репозиторий — немедленно отзови его:

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/mybots` → выбери своего бота
3. Нажми **API Token** → **Revoke current token**
4. Замени старый токен на новый в `.env`
